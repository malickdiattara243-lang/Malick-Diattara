##  Projets clés

###  Analyse des coûts d’assurance médicale (USA)

 *Analyse exploratoire et modélisation prédictive*  

- Nettoyage et analyse d’un **dataset réel d’assurance santé**
- Étude de l’impact de variables démographiques et de style de vie :
  - âge, sexe, IMC, tabagisme, nombre d’enfants, région
- Tests statistiques :
  - tests t, ANOVA, tests post-hoc
- Identification des facteurs majeurs :
  - **tabagisme, âge et IMC**
- Construction d’un **modèle de régression linéaire multiple**
  - ~ **75 % de variance expliquée**
- Visualisation :
  - boxplots, barplots
- Interprétation appuyée par la littérature scientifique

 **Outils** : Python, Pandas, NumPy, Matplotlib, Statsmodels
